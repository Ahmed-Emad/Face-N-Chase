# Face N Chase (Android Game)
link on Google Play:
  https://play.google.com/store/apps/details?id=com.ahmedemad.facenchase

###Description
A Game where a Smily Face is escaping from Angry Faces. You Should Help Him.

Face N Chase is a simple Game where a Smily Face is escaping from Angry Faces.

You should help him escape and avoid touching any of them.

Touch and Drag the Smily Face to escape from the Angry Faces.

-Note: If the Smily face is not moving, Touch then leave it before you Drag it.

Enjoy !
